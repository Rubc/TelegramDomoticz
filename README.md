# Telegram_Domoticz

## Getting started

* Install pyTelegramBotAPI using pip:

    sudo pip install pyTelegramBotAPI

## Create a Telegram Bot

* Replace <api_token> by the API token of your Telegram Bot
* Enter your Domoticz credentials

## Currently supported Telegram messages:

* Domoticz device ID followed by a command (On, Off, Stop) 
 
for example:
```1 On```
